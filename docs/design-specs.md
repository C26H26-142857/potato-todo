# 土豆ToDo - 设计规范

## 色彩方案

### 主色调
| 用途 | 色值 |
|------|------|
| 主色/强调 | #FFD60A（黄） |
| 习惯色1 | #FFD60A（活力黄） |
| 习惯色2 | #34C759（清新绿） |
| 习惯色3 | #FF6B6B（温暖粉） |

### 页面色彩
| 元素 | 浅色模式 | 用途 |
|------|---------|------|
| 页面背景 | #F5F5F7 | 全局背景 |
| 卡片背景 | #FFFFFF | 内容卡片 |
| 文字主色 | #333333 | 标题/正文 |
| 文字次色 | #999999 | 辅助信息 |
| 分割线 | #F0F0F0 | 列表分隔 |

### 今日页任务按钮
| 状态 | 背景 | 文字 |
|------|------|------|
| 未完成 | #F0F0F0 | #888888 |
| 已完成 | #FFD60A | #000000 |

### 倒计时
| 状态 | 数字颜色 |
|------|---------|
| 还剩 | #FFD60A |
| 已过去 | #CCCCCC |

### 日历热力图
| 密度 | 色值 |
|------|------|
| 无打卡 | 透明 |
| 少 | #FFF3B0 |
| 中 | #FFE066 |
| 多 | #FFD60A |

### 小组件
| 状态 | 卡片背景 | 图标/文字 |
|------|---------|----------|
| 未打卡 | #F5F5F5 | #999 / #555 |
| 已打卡 | #6B6B6B | #FFFFFF |
| 背景 | #F0F0F0 | - |

## 圆角规范
| 元素 | 圆角 |
|------|------|
| 卡片 | 14-16px |
| 任务按钮 | 20px |
| 小组件卡片 | 16-18px |
| 日期选中 | 12px |
| Tab 切换器 | 8px |

## 字体规范
- 全 App 使用系统默认字体（SF Pro）
- 标题：16-17px Bold
- 正文：13-15px Regular/Medium
- 辅助：11-12px Regular
- 大数字（倒计时/统计）：28-36px Bold
- 小标签：9-12px

## 小组件规格
- 网格：2 行 × 4 列
- 间距：8-10px
- 卡片圆角：16-18px
- 宽高比：3:4
- 内部布局：图标上 + 文字下，间距 8px，垂直居中
- 未打卡图标：深灰 360° 圆环（SVG 描边 2.5px）
- 已打卡图标：白色加粗 ✓

## 深色模式
- 支持自动/手动切换
- 页面背景、卡片、文字颜色跟随系统自动适配
- 小组件同步适配

## 动画
- 打卡完成：轻微缩放弹跳
- 日期切换：淡入淡出
- 列表操作：系统默认过渡
