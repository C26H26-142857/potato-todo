# 土豆ToDo — 习惯打卡 App 设计规格

## 概述

一个帮助用户养成好习惯的每日打卡工具。主色调黄色，支持深色模式。

## 技术栈

| 项 | 选型 |
|---|------|
| 框架 | SwiftUI |
| 数据 | SwiftData（本地存储） |
| 最低版本 | iOS 18+ |
| 小组件 | WidgetKit + App Intents |

## 数据模型

### Habit（习惯）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| name | String | 习惯名称 |
| sfSymbol | String | SF Symbols 图标名 |
| color | HabitColor | 预设色枚举（3色） |
| type | HabitType | single（单次）/ count（计数） |
| dailyTarget | Int | 计数型每日目标次数 |
| createdAt | Date | 创建时间 |

### CheckIn（打卡记录）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| date | Date | 打卡日期 |
| count | Int | 当日打卡次数 |
| habit | Habit | 关联习惯 |

### CountdownEvent（倒计时事件）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| name | String | 事件名称 |
| targetDate | Date | 目标日期 |

## 页面结构（4-Tab）

```
[今日]  [日历]  [统计]  [设置]
```

---

### 今日页（TodayView）

**顶部 — 日期选择器：**
- 固定显示 5 天（今天 ±2 天），今天默认居中
- 不可滑动，不可滚动查看更多
- 选中日期黄色高亮背景（#FFD60A），圆角 12px
- 日期上方星期简称，下方日期数字
- 今天上方显示"今天"而非星期

**中部 — 任务网格：**
- 2 列网格，动态行数（按习惯数量自动扩展）
- 每个按钮：20px 圆角药丸形
- 单次型习惯按钮：
  - 未完成：浅灰背景（#F0F0F0），深灰文字（#888），显示"○ 习惯名"
  - 已完成：黄色背景（#FFD60A），黑色文字，显示"✓ 习惯名"
  - 点击直接切换完成/未完成
- 计数型习惯按钮：
  - 未达标：浅灰背景（#F0F0F0），显示"习惯名 3/8"
  - 已达标：黄色背景，显示"✓ 习惯名"
  - 每次点击 +1，达到每日目标后自动变为完成状态

**底部 — 倒计时卡片：**
- 白色圆角卡片（16px），柔和阴影
- 标题："土豆ToDo · 未来已来"，加粗 16px，深灰，左对齐
- 项目垂直排列，项之间分割线分隔
- 每个项目左右结构：
  - 左：事件名称（加粗 15px 黑色）+ 目标日期（12px 灰色）
  - 右：标签"还剩"/"已过去"（11px 灰色）+ 大号数字（36px 加粗，未来=黄色 #FFD60A，过去=灰色 #ccc）+ "天"（11px 灰色）

---

### 日历页（CalendarView）

- 月份切换箭头（◀ 2026年X月 ▶）
- 7 列日历网格，星期头（日一二三四五六）
- 打卡日期按密度着色：
  - 浅黄 #FFF3B0（打卡少）
  - 中黄 #FFE066
  - 深黄 #FFD60A（打卡多）
- 底部图例：少 → 多
- 底部双卡片：当前连续天数 / 历史最长连续天数
- 点击日期可弹窗查看当天打卡详情

---

### 统计页（StatsView）

- 周/月分段切换器
- 完成率卡片（百分比大数字）
- 柱状图：每日打卡数，黄色柱，灰色当天无数据
- 底部双卡片：当前连续 / 历史最长

---

### 设置页（SettingsView）

**习惯管理：**
- 列表展示所有习惯（图标 + 名称 + 每日目标）
- 点击进入编辑页
- 底部"+ 添加新习惯"按钮
- 左滑删除

**倒计时管理：**
- 列表展示倒计时事件（名称 + 目标日期）
- 点击编辑
- 底部"+ 添加倒计时"按钮

**提醒：**
- 每日提醒开关 + 时间选择器（默认 08:00）
- 使用本地通知（UserNotifications）

**外观：**
- 深色模式开关

---

### 小组件（Widget）

**布局：**
- 2 行 × 4 列网格，间距 10px
- 每个卡片：圆角 18px，宽高比 3:4
- 卡片内部垂直居中：图标在上，文字在下，间距 8px
- 背景色：#F0F0F0

**未打卡状态：**
- 背景：#F5F5F5
- 图标：深灰（#999）360° 完整圆环（SVG 描边）
- 文字：深灰 13px

**已打卡状态：**
- 背景：#6B6B6B
- 图标：白色加粗 ✓
- 文字：白色 13px

**交互：**
- 点击卡片直接切换打卡状态（iOS 18 Widget Button + App Intents）

---

## 状态覆盖

| 状态 | 处理 |
|------|------|
| 无习惯 | 今日页网格为空，引导创建第一个习惯 |
| 全部完成 | 所有按钮变为黄色已完成状态 |
| 计数型习惯 | 每次点击 +1，按钮显示当前进度（如"3/8"），达到目标后变黄✓ |
| 过去日期 | 可点击查看但不能修改打卡状态 |
| 倒计时已过期 | 显示"已过去 X 天"，数字灰色 |
| 通知权限未授权 | 提醒开关关闭且不可用，引导开启权限 |
